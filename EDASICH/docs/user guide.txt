1. 	Purpose: 
	
	This document describes how to set up and configure EDASICH installation.

2. 	Concepts:
	
	

3. 	Installation MySQL:
	i. download and instal MySQL version 5.x
	ii. create new database name EDASICH
	iii. import database using file EPICS/EDASICH/db/EDASICH.sql 
	iv. change ip and port of external engines in table AnalyticEngine


4. 	Installation ESPERStreamProcessing:
	
	i. configure for esper at EPICS/Utils/ESPERStreamProcessing/src/main/webapp/WEB-INF/config.properties. Important configurations include MOM.IP, DB.EDASICH.IP, DB.EDASICH.USERNAME, DB.EDASICH.PASSWORD, SALAM.IP
	ii. compile Esper with maven2
	iii. deploy esper on Tomcat 7.x server


5. 	Installation EDASICH:
	
	i. configure for edasich at EPICS/EDASICH/src/main/webapp/WEB-INF/config.properties. The most important configurations are DB.EDASICH.IP, DB.EDASICH.USERNAME and DB.EDASICH.PASSWORD
	iii. Compiel edasich with maven2
	iv. deploy edasich on Tomcat 7.x server

	
6. 	Getting Started:
	
	i. Browse the web interface for data analytics of EDASICH at http://[ip]:8080/EDASICH/
	ii. Go to menu “DAF” to import/delete a data analytic function. Examples of data analytic function can be found in EPICS/EDASICH/examples/example1/daf3 
	iii. Go to salsa and turn on gas sensors http://128.130.172.215:8080/salsa-engine/?id=sensorsWorkload
	iii. After import a data analytic function. Go to menu “Analytics”. Choose streaming data. And click start to start analytic.
